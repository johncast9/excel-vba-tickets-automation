# Automatización de Tickets en Excel (VBA)

## Descripción
Este proyecto automatiza el procesamiento de una base de tickets en Excel usando VBA, generando datasets listos para su análisis posterior en Power BI.

## Funcionalidades
- Actualización incremental de base acumulada
- Limpieza y transformación de datos
- Generación de base de tickets
- Identificación de tickets pendientes

## Estructura del proyecto
- ActualizarBaseOriginal: actualización automática
- CrearBaseTickets: limpieza y transformación de los items
- CrearBasePendientes: filtrado de pendientes

## Flujo del proceso
1. Actualizar base acumulada
2. Crear base de tickets limpia
3. Generar pendientes

## Uso
1. Ejecutar macros en orden
2. Obtener bases listas para Power BI
